### Prerequisites
- Node.js (v14 or higher)
- Python 3.8+
- MongoDB
- OpenAI API key

### Installation
1. Clone the repository
2. Set up environment variables (see ENVIRONMENT_SETUP.md)
3. Install dependencies:
   ```bash
   # Backend
   cd backend
   pip install -r requirements.txt
   
   # Frontend
   cd frontend
   yarn install
   ```

### Running the Application
```bash
# Backend
cd backend
python server.py

# Frontend
cd frontend
yarn start
```
